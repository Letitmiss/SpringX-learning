# Redis的安装

## 安装单机Redis到linux系统

* 下载 : https://redis.io/download
* 解压  `tar -zxvf redis-4.0.12.tar.gz`
* 编译 ` cd  redis-4.0.12/ ; make` 
* 安装  `cd src/ ; make install `
* 启动服务端 `./redis-serve`
* 启动客户端 
  ```
  ./redis-cli 
  127.0.0.1:6379> set name cong
  OK
  127.0.0.1:6379> get name
  "cong"
  127.0.0.1:6379>
  
  ```
  * 守护进程方式启动 和bind ip
